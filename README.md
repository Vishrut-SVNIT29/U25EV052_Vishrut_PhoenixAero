## Project Overview

This repository contains the solutions to the Drone Programming Assignment completed using **Python** and **OpenCV**. The assignment covers fundamental Python programming concepts, object-oriented programming, recursion, computer vision, and basic drone-related applications.

The objective of this project is to understand how Python can be used in drone software development through practical programming exercises and theoretical concepts.
